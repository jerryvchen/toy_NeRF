# Toy NeRF
The goal of this project is to re-implement the original task of the NeRF paper. 

This is also my final project for the CS 117 (Projects in Computer Vision) class offered at UCI. 

## Tasks
1. Read NeRF
2. Implement NeRF
3. Calibrate own Images
